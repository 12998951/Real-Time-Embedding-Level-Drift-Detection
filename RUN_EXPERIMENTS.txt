Reproducing Experiments

1. Install dependencies
   pip install -r requirements.txt

2. Run synthetic drift test
   python drift_detector/test_drift.py

3. Run face embedding drift experiment
   python embeddings/face/test_face_drift.py

4. Run voice embedding drift experiment
   python embeddings/voice/test_voice_drift.py

5. Plot drift curves
   python utils/visualization/plot_drift.py
